Alert Cooldown
Suppress repeat alert notifications on this talkgroup for this many seconds after an alert fires. Useful for departments that always double-page. Set to 0 to disable (default).

Transcription Prompt
Custom prompt for Whisper / AssemblyAI transcription on this talkgroup. Overrides both the system-level and global prompts when set. Leave blank to inherit from the system or global prompt. For AssemblyAI, individual words are extracted and used as key-term boosts.

Linked Voice Channel (TGID)
Cross-channel paging: if this talkgroup transmits paging tones on a dedicated signalling channel but the voice dispatch happens on a different talkgroup, enter that other talkgroup's radio ID here. When tones fire on this channel the server will watch the linked channel for voice and attach the tones to it automatically. Leave at 0 to disable (default — no change to existing behaviour).

Voice Watch Window (seconds)
How long (in seconds) to listen on the linked voice channel after tones fire. Defaults to 30 seconds when left at 0.

Minimum Voice Duration (seconds)
Ignore voice calls on the linked channel shorter than this many seconds (filters out mic clicks and squelch tails). Leave at 0 for no minimum.
